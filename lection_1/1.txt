string - строка. "" - пустая строка
int - целые числа
uint - без знаковые значения. Кроме нуля
byte - байтовые
rune - символы
float, плавающие точки
complex - комплексные

var - объявление переменной
const - объявление констант. Константа - неизменяемое значение.

CTRL+D - скопировать целиком строку

%T - тип
%v - значение
%s - ??
\n - следующая строка
Вызов функции из вне - Название функции с большой буквы
= - присваивание
== сравнение

&& - логическое умножение. //И
|| - логическое сложение //ИЛИ

true && true = true
true && false = false
false && false = true

true || true = true
true || false = true
false || false = true

! - логическое отрицание:
!true = false
!false = true
